# Metric/Imperial Unit Converter

This is a Solo Project for Scrimba's JavaScript course. The app allows users to convert between metric and imperial units for length, volume, and mass.

## Features

- **Length Conversion:** Meters ↔ Feet
- **Volume Conversion:** Liters ↔ Gallons
- **Mass Conversion:** Kilograms ↔ Pounds
- All conversions are generated when the user clicks the **Convert** button.
- Results are rounded down to three decimal places.
- Includes a **Reset** button to clear the input and results.
- Responsive and user-friendly design following the provided spec.

## Usage

1. Enter a value in the input field.
2. Click **Convert** to see all conversions.
3. Click **Reset** to clear the input and results.

## Technologies

- HTML
- CSS
- JavaScript

## How to Run

1. Clone or download this repository.
2. Open `index.html` in your browser.

## Design Spec

- Clean layout with clear sections for input and results.
- Each conversion category displays both directions (e.g., meters to feet and feet to meters).
- Results are formatted to three decimal places.

## Example

If you enter `10` and click **Convert**:

- **Length:**  
  - 10 meters = 32.808 feet  
  - 10 feet = 3.048 meters
- **Volume:**  
  - 10 liters = 2.641 gallons  
  - 10 gallons = 37.854 liters
- **Mass:**  
  - 10 kilograms = 22.046 pounds  
  - 10 pounds = 4.535 kilograms
